# BlogMeiras

Projeto: Blog focado em notícias e conteúdo sobre a Sociedade Esportiva Palmeiras.
Stack: Next.js + TypeScript, Tailwind, Prisma (Postgres), NextAuth, OpenAI (summarização/embeddings), pgvector/Pinecone.

Roadmap inicial:
- Área pública (posts, notícias agregadas)
- Área admin (gerenciar posts, fontes, moderação)
- Agregador de notícias (RSS + scraper) com summarization via OpenAI
- Busca semântica com embeddings
- Deploy: Vercel + Supabase/Railway

Ver runbook no repo para configuração local.
